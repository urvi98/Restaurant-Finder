# Restaurant-Finder

//clone the code
Create a new folder
clone the code
open in vscode

//to run
open terminal
from root folder run command
npm i 

//backend
open two new terminals
in one terminal 
cd to backend
run npm i
and run npm start
also in index.js replace the api key with your own google map api key



//frontend
in another terminal 
cd to frontend
run npm i
and run npm start
also in App.jsx replace the api key with your own google map api key

make sure your location is on 
and hard refresh to start
